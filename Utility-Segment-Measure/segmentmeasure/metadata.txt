about=Utility Segment Measure is a QGIS plugin designed for utility locating, construction takeoffs, and field layout planning. The plugin provides real-time measurement feedback while digitizing features by displaying the previous segment length, live segment length, and running total distance of the active sketch. Unlike traditional measurement tools, Utility Segment Measure automatically synchronizes with QGIS digitizing operations, including vertex additions, deletions, sketch completion, and cancellation. The tool was developed to help translate plan-sheet measurements and PDF-based takeoffs into practical field-reference distances for locating, excavation, and construction workflows.
