# Changelog

## 1.0.1

### Fixed

- Resolved crash when undocking or floating the Utility Segment Measure panel.
- Removed dock visibility event handling that interfered with Qt dock widget state changes.
- Improved tracker shutdown handling.
- Improved dock close behavior.
- Prevented duplicate tracker instances from being created.

---

## 1.0.0

Initial public release.

### Features

- Real-time previous segment measurement.
- Real-time live segment measurement.
- Running total measurement.
- Floating HUD display.
- Unit conversion support (ft, m, in, mm).
- Toolbar toggle to start and stop the tracker.
- Exit Tracker button.
- Plugin-based deployment through QGIS.
